## Get Started

Clone the newly created repository and cd into it.
```shell
git clone <repository url> && cd <repository path>
```

<br>

Install the dependencies:
```shell
yarn
```
or
```shell
npm i
```

<br>

Build css and js:
```shell
yarn run build
yarn run scss
```
or
```shell
npm run build
npm run scss
```

<br>

Open `build/index.html` in browser
